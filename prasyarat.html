    Prasyarat Sistem

    Berikut kebutuhan sistem dan persiapan sebelum memulai tutorial:

    Kebutuhan Perangkat Lunak
    KomponenVersiDeskripsi
    PostgreSQL 17.5 Database server utama
    Windows 11 Sistem operasi (64-bit)
    VSCode Terbaru Editor kode dengan ekstensi PostgreSQL
    Konfigurasi Database
    Host: localhost Port: 5432 User: postgres Password: P@ssw0rd123! Database: test_db
    Instalasi Ekstensi VSCode
    1. Buka VSCode, tekan Ctrl+Shift+X
    2. Cari "PostgreSQL" (oleh Chris Kolkman)
    3. Klik Install
    4. Konfigurasi koneksi dengan detail di atas
    Struktur File CSV

    File harus tersedia di C:\data\ dengan format:

    conservation_id,location,area_ha,carbon_credits,date_recorded C001,Aceh Jaya,50,250,2024-01-15

    PERSIAPAN AWAL

    Verifikasi Instalasi

    Jalankan SELECT version(); untuk memastikan PostgreSQL 17.5 terinstal.

    Alternatif SQLTools

    Jika ekstensi utama bermasalah, gunakan SQLTools oleh Matheus Teixeira.

    Folder Data

    Buat folder C:\data\ dan pastikan PostgreSQL memiliki akses baca.
