Pendahuluan

Tutorial ini melanjutkan panduan instalasi PostgreSQL 17.5, berfokus pada pembuatan dan pengelolaan database konservasi mangrove menggunakan data dari file CSV (001Mangrove_Conservation_Records.csv, dll.).

Tujuan Tutorial
  1. Membekali peserta dengan keterampilan manajemen database PostgreSQL untuk konservasi mangrove
  2. Memahami integrasi sistem database dengan teknologi blockchain
  3. Menyediakan panduan langkah demi langkah dari dasar hingga lanjutan
